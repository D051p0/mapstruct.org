Getting help

You’are using MapStruct and ran into a problem?

First check out the reference guide. If that doesn’t help answering your question you may join the MapStruct Google group.

To report a bug or request a new feature use the MapStruct issue tracker. Note that bug reports should be accompanied by a test case demonstrating the issue.
