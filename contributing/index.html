Contributing

You love MapStruct but miss a certain feature? You found a bug and want to report it, or even better, fix it yourself? That’s great! MapStruct is all open source and your help is highly appreciated.

WhatWhere
Source codehttp://github.com/mapstruct/mapstruct
Issue trackerhttps://github.com/mapstruct/mapstruct/issues
DiscussionsJoin the mapstruct-users Google group
CI buildhttps://mapstruct.ci.cloudbees.com
This web sitehttp://github.com/mapstruct/mapstruct.org

MapStruct follows the Fork & Pull development approach. To get started just fork the MapStruct repository to your GitHub account. Once you are done with your change, submit a pull request against the MapStruct repo.

This web site (mapstruct.org) is created using the fantastic awestruct tool. To update this site, fork the mapstruct.org repo and create a pull request. The individual pages are written in Haml and Markdown. Building the site using awestruct requires Ruby to be installed on your system.
